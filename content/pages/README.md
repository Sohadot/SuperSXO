# content/pages/

## Governance Rule

This directory holds page source content for SuperSXO.com.

No page may be drafted, placed here, or published unless:

1. The route is registered in `data/routes.json`
2. The route's `status` is exactly `published`
3. The route has passed every quality gate defined in `data/quality-gates.json`
4. The publication is logged in `DECISION_LOG.md`

## Directory Convention

Page content files should follow the route path structure.

Example: the route `/what-is-sxo/` maps to `content/pages/what-is-sxo.md`

## Current Status

No routes have status `published`.
No page content files may be created yet.

Content files will be added here only when the corresponding route advances
from `planned` to `drafted` to `approved` to `published` through the
governed workflow.
