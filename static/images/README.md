## Image Standards

All images added to this directory must follow these rules:

1. **No stock imagery by default.** Generic stock photos contradict the institutional, precise character of the Sovereign Spatial Interface.

2. **No heavy decorative assets.** Images that serve only decoration and carry no semantic meaning are prohibited.

3. **All meaningful images require `alt` text.** Purely decorative images must use `alt=""` and `role="presentation"`.

4. **Images must support the spatial diagnostic interface direction.** Assets should reinforce the institutional, analytical character of SuperSXO — not decorate it.

5. **Performance constraint.** All images must be optimized before deployment. No image may block rendering. Use appropriate formats (WebP preferred, AVIF where supported).

6. **No affiliate images, no partner badges, no third-party logos** without explicit approval and a `DECISION_LOG.md` entry.
